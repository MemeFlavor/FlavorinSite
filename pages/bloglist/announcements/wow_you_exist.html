                    Pretty awesome right? Anyways, so now I'll be explaining the stuff I build this site from the ground up. One thing you'll notice on
                    the first paragraph of this blog, I've used a thing called "SCSS" it's basically an extension of CSS that makes coding in
                    CSS more "bearable" for me personally. It has nesting and variables (already supported natively but acts differently),
                    mixins, modularity, and some advanced stuff.
               

               

                    If you're interested in learning this, considered some key consideration. If you're new to coding pls learn the
                    fundamentals of CSS first, so you can understand what the fuck you're actually doing. And If your website is really
                    small and simple just use CSS, it's overkill and unnecessary. Use it, if your site is big and complicated that it actually
                    requires modularity to make it more maintanable in the long run.
               

               

                    And another big consideration is that you have to download it, it's not simple as clicking the download button. You have to use the
                    terminal if you want it. So if that bothers you, you should probably stick with CSS. And for those who are brave enough to do 
                    this, I commend you, you crazy mf.
